Objetivo do Projeto

Criar uma aplicação web que usa a câmara do dispositivo para reconhecer gestos da Língua Gestual Portuguesa (LGP) e os traduz para texto em tempo real. O modelo de reconhecimento é treinado pelos próprios alunos no browser. A aplicação inclui síntese de voz integrada, construtor de frases interativo e um servidor Python (Flask) para guardar e gerir de forma permanente o histórico de tradução.

Distribuição de 15 horas
F1
Descoberta e Contexto 2h

Antes de programar, perceber o propósito. A LGP é uma língua completa com gramática própria.

  • Ver um vídeo curto sobre a LGP e a comunidade surda em Portugal
  • Conhecer o alfabeto manual português (dactilologia) — 26 letras e os dígitos
  • Decidir o subconjunto a traduzir: as letras A–Z ou 10 palavras simples
  • Criar uma conta Google para a Teachable Machine
  • Explorar a interface de treino
💡 Recomendação: Começar com apenas 5 letras (A, B, C, D, E) nas primeiras sessões.
F2
Recolha de Dados e Treino 3h

Esta é a fase crítica. Um modelo só é eficaz com dados diversificados.

  • Aceder a teachablemachine.withgoogle.com
  • Criar uma classe específica por letra/palavra
  • Recolher **no mínimo 50 fotografias** com a webcam do browser
  • Variar fatores: focar em fundos diferentes, iluminação e mãos
  • Adicionar uma classe "Nenhum" com imagens neutras (essencial!)
  • Clicar em Train Model e aguardar (2 a 5 minutos)
  • Testar exaustivamente no preview da plataforma
⚠️ Erro comum: Capturar fotografias sempre no mesmo plano de fundo. Diversifique!
F3
Exportar Modelo e Conectar 2h

Gerar o link no Teachable Machine e ligar ao tradutor mobile.

  • Na plataforma: Export Model → Tensorflow.js → Upload (shareable link)
  • Copiar o URL gerado
  • Ir ao painel do "Tradutor" desta app
  • Colar o link e premir "Carregar Modelo"
  • Premir "Iniciar Câmara" para testar
F4
Melhorar e Personalizar 5h

Com as deteções básicas a funcionar, aprofunde a aplicação.

  • Adicionar novas palavras ou expressões ao modelo
  • Personalizar o estilo visual e as cores do tema
  • Construir um histórico de deteções interativo
  • Ajustar o limiar de sensibilidade (Threshold)
F5
Publicar e Apresentar 3h

Partilhar o seu tradutor publicamente online e apresentá-lo.

  • Publicar os ficheiros estáticos gratuitamente via GitHub Pages
  • Testar no telemóvel: guardar o link no ecrã principal como PWA
  • Preparar uma breve apresentação de 3 minutos cobrindo: Problema → Solução → Demonstração

Criação de Modelos Passo a Passo
  • Abra teachablemachine.withgoogle.com no seu telemóvel ou computador.
  • Selecione "Get Started" → "Image Project" → "Standard image model".
  • Renomeie as classes para representar a letra ou gesto pretendido (ex: clique em "Class 1" e digite "A").
  • Prima o botão "Webcam" e mantenha pressionado o botão de captura para tirar imagens dos gestos (mínimo 50).
  • Adicione obrigatoriamente uma classe com o nome "Nenhum" e capture fotos neutras (apenas do fundo).
  • Concluídas as classes, clique no botão central "Train Model".
  • Teste o desempenho do reconhecimento no painel de "Preview" à direita.
  • Estando funcional: selecione "Export Model" → "Tensorflow.js" → "Upload (shareable link)" e copie o link.
  • Insira esse link no painel "Tradutor" desta aplicação e clique em carregar.

Recomendações para Obter Maior Precisão
  • Volume de dados: Tente obter no mínimo 50 capturas por gesto.
  • Variação de cenários: Registe fotos alterando a distância do ecrã, a iluminação e os planos de fundo.
  • A importância do "Nenhum": Esta classe evita que o modelo tente adivinhar gestos quando não está ninguém à frente da câmara.
